================================================================================
                    ANIME-MINER SCRIPT CATALOG & EXPLANATIONS
================================================================================

This catalog explains the exact functionality of each miner script inside the 
`anime-miner` directory. These scripts are used to scrape, extract, and save 
streaming video paths or torrent download links to the Supabase database.

--------------------------------------------------------------------------------
1. miner.js
--------------------------------------------------------------------------------
- Target Domain: https://gogoanime.es
- Type: HTTP Streaming Links
- Purpose: 
  Automatically checks the homepage timeline of Gogoanime.es for recently 
  released episodes, tracks backwards page-by-page, extracts the embedded 
  video players (like GogoCDN, Vidstreaming), and saves the iframe URLs to Supabase.
- Cache Management:
  Tracks progress in `visitedSeries.json` to avoid scraping already-processed series.

--------------------------------------------------------------------------------
2. miner_http.js
--------------------------------------------------------------------------------
- Target Domain: https://gogoanime.or.at
- Type: HTTP Streaming Links
- Purpose:
  Deep-dives into a specific anime by searching Gogoanime.or.at. Can be run 
  with a command-line argument (e.g. `node miner_http.js "one-punch man"`). 
  It crawls the search results, lists all episodes, extracts the compatible 
  streaming player iframes (Sub and Dub), and uploads them to Supabase.
- Cache Management:
  Uses `global_cache.json` to keep track of the highest mined episode for 
  cached series so that subsequent runs skip already-saved episodes.

--------------------------------------------------------------------------------
3. miner_gogo_or_at_https_dump.js
--------------------------------------------------------------------------------
- Target Domain: https://gogoanime.or.at
- Type: HTTP Streaming Links
- Purpose:
  Similar to `miner.js`, but targets the timeline of `Gogoanime.or.at`. It is 
  configured as a timeline "dump" miner which crawls backwards from Page 1 to 
  discover new releases and backfill the database with streaming iframe links.
- Key Configuration:
  Optimized with Puppeteer Stealth and Adblocker plugins. Configured to wait for 
  `networkidle2` to ensure client-side JS finishes rendering the dynamic players.

--------------------------------------------------------------------------------
4. miner_iframe.js
--------------------------------------------------------------------------------
- Target Domain: Configured via NINEANIME_URL (.env) - (e.g. https://9anime.tech)
- Type: HTTP Streaming Links
- Purpose:
  Targets 9Anime/Aniwave clones. It scouts the home timeline for recently added 
  episodes, extracts the direct player iframes (filtering out tracking scripts 
  like ShareThis), and saves them to the database.

--------------------------------------------------------------------------------
5. miner_nyaa_anime.js
--------------------------------------------------------------------------------
- Target Domain: https://nyaa.si (Anime Category: 1_2 - English-translated)
- Type: P2P Torrent Magnet Links
- Purpose:
  Scrapes the latest anime uploads on Nyaa.si. It parses the torrent titles to 
  extract the clean anime names and episode numbers, then saves their Magnet URLs 
  to Supabase as `torrent` type links. These allow users to play decentralised 
  torrents directly in their client (like WebTorrent).

--------------------------------------------------------------------------------
6. miner_nyaa_manga.js
--------------------------------------------------------------------------------
- Target Domain: https://nyaa.si (Manga Category: 3_1 - Literature English-translated)
- Type: P2P Torrent Magnet Links
- Purpose:
  Scrapes the literature section of Nyaa.si to index translated manga. It parses 
  manga volume/chapter titles and saves their Magnet links to Supabase for manga 
  reading/download integration.
